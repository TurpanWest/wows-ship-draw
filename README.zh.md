# WoWs Ship Draw

这是一个为《战舰世界》玩家设计的抽船工具——如果你对着港口里一堆船不知道今天玩什么，那就随机抽一艘吧！

**支持英文和中文双语！**

---

## 功能

- **内置 T10 舰船列表**：覆盖驱逐舰（DD）、巡洋舰（CA）、战列舰（BB）、航空母舰（CV）和潜艇（SS）。所有巡洋舰统一标记为 CA，方便筛选。
- **自定义列表**：可手动添加舰船，输入名称并选择舰种即可。
- **舰船管理**：
  - 取消勾选 → 临时排除该船（比如今天不想玩这艘）。
  - 点击 ❌ → 从列表中永久删除（比如你没有这艘船）。
- **排除舰种**：勾选"排除 CV"或"排除 SS"可将航母或潜艇从随机池中移除。
- **等级筛选**：切换哪些等级（VIII / IX / X / XI）参与抽取。
- **国籍筛选**：13 个游戏国籍的多选标签，支持全选/全不选。
- **恢复默认**：点击"恢复默认"可将抽取池重置为预设舰船列表。
- **多语言支持**：通过右上角语言按钮切换英文和中文。

## 使用方式

在线访问：https://wows-ship-draw.vercel.app/
